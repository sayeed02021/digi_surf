# Digi-Surf: A Python package for generating non-ionic surfactants

## Package Installation Steps
1. Clone the repository
```
git clone https://github.com/sayeed02021/digi_surf.git
cd digi_surf
```
2. Create a new `conda` environment:
```
conda create -n digi_surf python=3.10
conda activate digi_surf
```
3. Install necessary libraries and files:
```
pip install .
```

**NOTE:** Package has been tested to run successfully on Linux systems equipped with NVIDIA GPU. If your system does not support CUDA,then diffusion model will not run.
